UserPlat

字段名 说明 默认值
id 平台id
pname 平台名称

UserInfo

字段名 说明 默认值
pid 外键关联UserPlat的id
uid 对应平台的userid，例如appuserid,openid,指纹id和其他第三方的userid
nickname 昵称
headimgurl 头像
sex 性别（0未知，1男，2女）

GameModel

字段名 说明 默认值
id 主键自增
title 游戏原型标题
description 游戏原型描述（游戏所需的元素，注意事项等）
model 游戏需满足的配置项json字符串形式存储 
消消乐游戏在GameModel表中的model字段如下：
{{dataSource|json}}

GameEntity

字段名 说明 默认值
id 主键自增
gid 外键关联GameModel的id（一个游戏原型可以创建多个游戏实例）
title 游戏SEO之标题
keywords 游戏SEO之keywords
description 游戏SEO之description
name 游戏名称
linkurl 游戏链接（根据游戏名称生成，只读）
iconurl 游戏icon图标链接

ImageOptions

字段名 说明 默认值
id 主键自增
geid 外键关联GameEntity的id
imgurl 图片路径，上传后生成的
description 图片描述（做什么，用在什么地方）
pixel 建议像素分辨率（类似800*600，用于给图片维护时的提示）

AudioOptions

字段名 说明 默认值
id 主键自增
geid 外键关联GameEntity的id
audurl 声音文件路径，上传后生成的
description 声音文件描述（用途，出现时机）

Score

字段名 说明 默认值
id 主键自增
pid 外键关联UserPlat的id
uid 对应平台的userid，例如appuserid,openid,指纹id和其他第三方的userid
geid 对应GameEntity的id
value 得分
datetime 提交分数时的时间戳

Rank

字段名 说明 默认值
id 主键自增
sid 外键关联Score的id
ranking 排名
headimgurl 用户头像（可通过sid中找到Score中标识的用户查找显示，这里作为冗余字段）
nickname 用户昵称（可通过sid中找到Score中标识的用户查找显示，这里作为冗余字段）
